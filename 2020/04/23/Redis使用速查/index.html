Redis使用速查

key-value

  • set key vlaue 设置键值对
  • get key 得到键值对
  • del key 删除键值对
  • expire key second 设置键值对的过期时间
  • flushall 删除数据库所有键值对

key-member

  • sadd key member 设置键数组
  • smembers key 得到键数组
  • sismember key value 查看value是否在key中
  • scard key 查看key中元素个数
  • srem key value 删除key中的value元素
  • spop key 从key中取出一个值，并从key中删除
  • srandmember key 从key中取出一个值
  • incrby key -1 将键值减一
