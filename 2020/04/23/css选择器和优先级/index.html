css选择器和优先级

选择器

css选择器用来选择HTMl当中的元素，来给元素添加相应的样式。分别分为简单选择器，属性选择器，伪类选择器，伪元素选择器和组合选择器。

通用选择器

通用选择器（*）可以选择整个页面的所有元素。使用方法如下：

类选择器

类选择器的形式为“.ClassName”，即在类名之前加一个“.”。

ID选择器

ID选择器的形式为“#ID”，即在ID前加一个“#”，值得注意的是一个页面，相同的ID应该只有一个。

ID选择器和类选择器的区别

ID选择器在一个页面通常只对应一个元素，而类选择器则对应则是许多元素，所以一般一个页面只出现一次的用ID像网站的头部，主体和脚部，而多次出现的就采用类。

属性选择器

属性选择器选择的是具有某种属性的元素，用[]代表，[]内为要选择的属性。

伪选择器

伪类选择器

伪类选择器是指选择元素的某个状态，部位或者在特定情况下的元素。常用的有：

  • :link 未访问的链接。
  • :visited 已访问的链接。
  • :hover 鼠标悬停在元素上时。
  • :active 当鼠标按下和松开时。
  • :first-child 一组元素中的第一个元素
  • :first-of-type 一组元素中的第一个type元素
  • :nth-child(an+b) 匹配对应位置的元素，0n+1就是第一个，2n+0就是2、4、6…这些元素。
  • :nth-of-type(an+b) 一组元素中的第an+b个type元素。
  • :focus 当元素获得焦点。

    值得注意的是，当使用link、visited、hover、actived时要按照LVHA的顺序，即link->visited->hover->actived。

伪元素选择器

伪元素选择器选择元素的某个部分。常用的有：

  • ::after 在元素的后面创建一个伪元素，必须包含content属性。
  • ::before 在元素的前面面创建一个伪元素，必须包含content属性。
  • ::first-letter 选择第一行的第一个字母。
  • ::first-line 选择第一行。

组合选择器

  • A,B:包含A元素或B元素的任意元素。
  • A B:选择B元素且B元素是A元素的后代。
  • A>B:选择B元素且B元素是A元素的直接后代。
  • A+B:选择B元素且B元素是A元素的下一个直接相邻元素。
  • A~B:选择B元素且B元素是A元素之后的任意元素

优先级

选择器优先级的顺序是：

  1. !important
  2. 内联元素
  3. ID选择器
  4. 类选择器 属性选择器 伪类选择器
  5. 标签选择器 伪元素选择器
  6. 通配选择器

当一个元素同时被多个选择器选择时，记ID选择器的总数为A，类选择器、属性选择器、伪类选择器的总数为B，标签选择器、伪元素选择器总数为C。按A、B、C顺序依次比较，数量越多则优先级越大，当A、B、C数量相等时，则在下面的选择器会覆盖上面的选择器。
