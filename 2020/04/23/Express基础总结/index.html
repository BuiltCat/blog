使用express搭建服务器

Express 基础

通过express搭建服务器

express中间件

body-parser

Request

  1. req.params:识别定义路径时以冒号开头的参数
  2. req.body:经过body-parser转码后的body对象
  3. req.method:请求方法
  4. req.query:经过node原生querystring或者qs库识别的http query “http://localhost:3200/?name=123&age=321"
  5. req.get:获取header

Response

  1. res.send:发送内容
  2. res.append:在请求头中加入内容
  3. res.redirect:重定向请求
  4. res.json:发送JSON给客户端
